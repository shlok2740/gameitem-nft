**NOTE**
Firstly, Create a .env file and add two variables
- QUICKNODE HTTP URL
- YOUR PRIVATE KEY

Steps -

- Create a NFT Solidity smart contract.
- Deploy the smart contract on Goreli Network
- Verify on Goreli Etherscan
